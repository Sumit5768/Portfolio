Projects

ATM Simulation

This project is a simple ATM simulation system built using Python. It allows users to log in securely using an account number and PIN, with a maximum of three attempts. User data, including account number, PIN, and balance, is stored and managed using a CSV file. After successful login, users can check balance, deposit money, withdraw funds, and change their PIN. All transactions are updated in real time and saved back to the file. This project demonstrates concepts like file handling, object-oriented programming, and basic banking operations..

Student Result Analysis

This project performs data analysis on student performance using Python libraries like Pandas, Seaborn, and Matplotlib. It involves data cleaning by handling missing values and removing duplicates to ensure accurate results. The project calculates key insights such as average study hours, average marks, highest scoring student, and lowest attendance. It also analyzes relationships between variables using correlation techniques. Data visualization is done through bar charts, histograms, and scatter plots to better understand patterns. Overall, this project demonstrates data preprocessing, analysis, and visualization skills in a practical scenario.
